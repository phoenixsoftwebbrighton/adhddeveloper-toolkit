// the toolkit

Three tools. One mission.

🗣️
GitSpeak
git without the syntax

Type one word. GitSpeak handles the rest. Stage, commit, push — in plain English. No more forgetting the right flags.

$ gs
→ save — stage + commit + push in one go
→ undo — safely undo last commit
→ status — see what's changed
🔍
Decipher
errors in plain english

Paste any error message. Get a plain English explanation, what caused it, and exactly how to fix it. No Stack Overflow required.

$ decipher
→ paste error — get plain English back
→ community — submit unknown errors
⚒️
ansible-forge
infrastructure without the overwhelm

Scaffold Ansible roles in seconds. Type 3 letters, pick a service, watch the boilerplate build itself. No more staring at blank YAML files.

$ forge
→ pih — scaffold pihole role
→ doc — scaffold docker role
